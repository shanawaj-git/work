# [Backstage](https://backstage.io)

This is your newly scaffolded Backstage App, Good Luck!

To start the app, run:

### Env variables

```sh
BACKSTAGE_DB_HOST=
BACKSTAGE_DB_PORT=
BACKSTAGE_DB_USER=
BACKSTAGE_DB_PASSWORD=
BACKSTAGE_HOST=localhost
BACKSTAGE_PORT=3000
```

```sh
yarn install
yarn dev
```
